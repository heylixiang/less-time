# less-time ⏱️
A lightweight, modern countdown timer built with React + Vite, designed for distraction-free time management.

---

## ✨ Key Features
- 🚀 Blazing-fast performance powered by Vite, with near-instant load times
- 🎨 Minimal, clean UI focused solely on countdown functionality, no ads or bloat
- 📱 Fully responsive design that works seamlessly on desktop, tablet, and mobile
- 🌍 One-click deployment via GitHub Pages, ready to use out of the box
- 🧪 Automated testing ready with Playwright integration for reliable UI validation
- 🛠️ Built with TypeScript for type safety and maintainable code

---

## 🛠️ Local Development
### Prerequisites
- Node.js 18+
- npm/yarn/pnpm

### Installation & Run
```bash
# Clone the repository
git clone https://github.com/heylixiang/less-time.git
cd less-time

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
